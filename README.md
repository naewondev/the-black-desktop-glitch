Black Desktop Glitch Simulator
-------------------------------
Replicates the classic Windows black desktop glitch.

HOW TO USE:
1. Double-click black_desktop_glitch.exe
2. Your desktop will go black
3. Click and drag to reveal your desktop
4. The program closes automatically when fully revealed
5. Press ESC at any time to exit
6. Have fun!

SAFETY:
- No internet connection
- No files are created or modified
- No background processes
- Windows only
